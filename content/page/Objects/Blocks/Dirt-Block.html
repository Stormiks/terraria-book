Земляной блок (Dirt Block) — естественный блок, один из самых распространённых в игре. Представляет собой "почву", на которой в большинстве поверхностных биомов растут трава и деревья. Также в относительно небольших количествах находится в каменном слое и ниже. Не может быть использован для выращивания растений, пока в своем первоначальной виде.

Область распространения

  • Лес: встречаются повсеместно, составляют большую часть ландшафта.

  • Подземелье, землянной слой: по сравнению с предыдущим биомом, удельное содержание земляных блоков уменьшено, но они по прежнему составляют большую часть слоя.

  • Подземелье, каменный слой: по прежнему часто встречаются, но уже не составляют большую часть биома.

  • Версия 1.1:Теперь может быть использован для создания блоков грязи, когда персонаж находится рядом с водой..

  • Версия 1.0.4:Теперь графически "сливается" с грязью.
